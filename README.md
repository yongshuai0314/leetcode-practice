# LeetCode 刷题记录

这个仓库用于记录我的 LeetCode 刷题过程。

## 目录结构
- src/: 源代码目录
  - arrays/: 数组相关题目
  - strings/: 字符串相关题目
  - linkedlists/: 链表相关题目
  - trees/: 树相关题目
  - dynamicprogramming/: 动态规划相关题目
  - graphs/: 图相关题目
  - utils/: 工具类

## 使用方法
1. 克隆仓库
2. 使用 Maven 构建项目: `mvn clean install`
3. 运行测试: `mvn test`
